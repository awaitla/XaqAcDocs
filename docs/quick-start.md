# 快速开始

## 工作流
每个班级**第一次使用**本工具时执行**完整工作流**，**之后只需**执行**简化工作流**，因为“重置密码>修改密码”这个部分是可持续的。

**完整工作流：**
获取学生账号列表>重置密码>修改密码>开始答题

**简化工作流：**
获取学生账号列表>开始答题
